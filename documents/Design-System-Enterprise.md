# THEESMA SPORTS — DESIGN SYSTEM (ENTERPRISE)

## 1. DESIGN TOKENS

### Colors
- Primary: #000000
- Background: #FFFFFF
- Text: #1A1A1A
- Accent Blue: #1E90FF
- Accent Orange: #FF6A00
- Gold Accent: #D4AF37

### Usage
- Primary for headers
- Accent for CTAs
- Gold for premium highlights

---

## 2. TYPOGRAPHY

### Headings
- Font: Inter / Poppins
- Weight: 600–800
- Style: Uppercase for hero

### Body
- Font: Inter
- Weight: 400–500

---

## 3. SPACING SYSTEM

- 4px base grid
- 8 / 16 / 24 / 32 / 48 / 64 spacing scale

---

## 4. COMPONENT SYSTEM

### Buttons
- Primary: Filled (accent color)
- Secondary: Outline
- States: hover, active, disabled

### Cards
- Product card
- Event card
- Elevated with shadow

### Forms
- Input fields with labels
- Error + success states

---

## 5. GRID SYSTEM

- Desktop: 12 column grid
- Tablet: 8 column
- Mobile: 4 column

---

## 6. MOTION SYSTEM

### GSAP Rules
- Duration: 0.6–1.2s
- Easing: power3.out
- Use scroll triggers for storytelling

---

## 7. 3D GUIDELINES

- Use Spline for hero products
- Lazy load models
- Provide fallback images

---

## 8. ADMIN PANEL DESIGN

### Layout
- Sidebar navigation
- Top header (search + profile)

### Modules UI
- Dashboard: KPI cards
- Products: Table + filters
- Orders: Status pipeline
- Inventory: Stock alerts
- Events: Content editor

### Table Design
- Sortable columns
- Pagination
- Search

### Forms
- Multi-step for products
- Variant management UI

---

## 9. UX PRINCIPLES

- Mobile-first
- Fast interactions
- Clear hierarchy
- Minimal friction checkout

---

## 10. FINAL RULE

Design must drive conversion + trust.
